# Chip-8 Emulator

Реализация логического ядра эмулятора Chip-8 на Go.
Проект создан для изучения низкоуровневой работы с памятью и процессором.

## Статус проекта

*   **CPU & Memory:** Логика работы регистров, стека и памяти реализована.
*   **Graphics & Input:** Вывод графики и обработка ввода в процессе разработки.
